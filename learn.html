Flex properties

If we are using flex as display property we can't use text-align in that case we have justify-content property, which works same(align text inside div center horizontally) and align-item property (which keeps inside div things in center vertically)

to align vertically without flex we can use line-height same as height but only for signle line .. if we have multiple lines we have to give these three properise (display:inline-block, line-height:normal, vertical-align:middle)

absolute- relative to dom sets top bottom left right relative- relative to its actual position sets top bottom left right transform- propertise(transform, scale, rotate, skew, initial, inherit)
